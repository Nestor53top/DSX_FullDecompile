# DSX Full Decompile

Automatic decompilation of **DualSenseX (DSX)** binaries using [ilspycmd](https://github.com/icsharpcode/ILSpy).

## What is this?

DualSenseX is a Windows application that brings PS5 DualSense controller features (Adaptive Triggers, Haptic Feedback, LED) to PC games. The app is written in **C# (.NET 4.5)** and distributed as a Squirrel installer. This repository contains:

1. **Original binaries** (`binaries/`) — installers and drivers
2. **Decompiled source** (`decompiled/`) — auto-generated C# code from the .NET assemblies
3. **Tools** (`tools/`) — dnSpy for manual debugging/analysis

## Tools

### ilspycmd (used in GitHub Actions)
The primary decompiler. Cross-platform CLI tool for .NET assemblies.

### dnSpy (manual analysis, Windows only)
Located in `tools/dnSpy/`. Downloaded automatically by GitHub Actions, or manually from [dnSpy releases](https://github.com/dnSpy/dnSpy/releases).

Features:
- Decompile with syntax highlighting
- Edit and recompile C# code
- Debug .NET applications
- Search across all assemblies

## How it works

1. GitHub Actions installs .NET 8 SDK + ilspycmd
2. Downloads DualSenseX installer from GitHub releases
3. Extracts Squirrel package with 7zip (NuGet-like format)
4. Runs `ilspycmd -p -o decompiled/<name> <assembly.dll>` on each .NET assembly
5. Commits all `.cs` files back to the repo
